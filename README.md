# unitree-g1-auto-charger-plugging
Kompletne oprogramowanie dla Unitree G1 EDU do autonomicznego ładowania pojazdów. Robot podchodzi do stacji, pobiera ładowarkę, a następnie nawiguje do auta i precyzyjnie umieszcza wtyczkę w porcie. Projekt obejmuje pełny proces: od nawigacji i wizji komputerowej po zaawansowaną manipulację ramionami humanoida.
